How It Works

Score is built from 4 categories, each worth 25 points:

25 Content Density (overall + first screen + consistency)
25 White Space (real element gaps)
25 Complexity (items per screen)
25 Readability (font, spacing, contrast)
<40 Bad 40+ Dense 55+ Okay 70+ Good 85+ Great

Tracks attention using 3 simple signals:

Mouse Proximity

Follows cursor movements closely.

Dwell Time

Increases when looking. Hovers double it.

Scroll Speed

Fast scrolling lowers the tracking score.

Items heat up at different speeds based on type:

High Buttons & Links
Med Headings & Text
Low Small Details
Content Density

Content coverage. Checks overall, first screen, and consistency.

White Space

Real pixel gaps between elements. Ideal: 20–48px.

Complexity

Average items per view. Ideal: 8–15.

Readability

Font size, line height, line length, and contrast.
